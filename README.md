## 아쉬운 점
- MVVM 기반에서 명확하게 View 와 ViewModel 의 역할을 나누지 못한 점
- 공통적으로 쓰이는 footer section 을 모듈화하지 못한 점
- 네트워크 통신에서 발생할 수 있는 예외 상황에 대한 방어 코드가 미흡한 점
